Security Notice: Clear Saved Passwords

For enhanced security, please clear any saved passwords for this payroll system

Important: This payroll system contains sensitive employee and financial data. Saved passwords in browsers can pose a security risk, especially on shared computers.

How to Clear Saved Passwords by Browser:

Google Chrome

  1. 1Click the three dots menu (⋮) in the top-right corner
  2. 2Go to Settings
  3. 3Click Autofill in the left sidebar
  4. 4Click Password Manager
  5. 5Find entries for this payroll system and click the trash icon to delete

Mozilla Firefox

  1. 1Click the menu button (☰) in the top-right corner
  2. 2Go to Settings
  3. 3Click Privacy & Security in the left panel
  4. 4Under "Logins and Passwords", click Saved Logins
  5. 5Find entries for this payroll system and click Remove

Safari

  1. 1Go to Safari menu → Preferences
  2. 2Click the Passwords tab
  3. 3Authenticate with Touch ID or password
  4. 4Find entries for this payroll system and click Remove

Microsoft Edge

  1. 1Click the three dots menu (...) in the top-right corner
  2. 2Go to Settings
  3. 3Click Profiles in the left sidebar
  4. 4Click Passwords
  5. 5Find entries for this payroll system and click the trash icon
Additional Security Tips:
  • Always log out completely when finished
  • Never save passwords on shared or public computers
  • Use strong, unique passwords for the payroll system
  • Report any suspicious activity to your system administrator
